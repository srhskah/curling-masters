配置前后端分离架构

📡 后端配置

🗄️ Turso 数据库配置

在 Zeabur 服务中设置以下环境变量：

DATABASE_TYPE=turso
TURSO_URL=your-turso-database-url
TURSO_AUTH_TOKEN=your-turso-auth-token
SECRET_KEY=your-secret-key
DB_ENCRYPTION_KEY=your-encryption-key

🌐 前端部署配置

修正前端代码中的 API 地址：

// 在 frontend/index.html 中修改：
const API_BASE = '请在此输入您的 Zeabur 后端地址';

✅ 测试步骤

  1. 在 Zeabur 设置 Turso 环境变量并重新部署
  2. 测试 /api/turso-test 端点确认数据库连接
  3. 更新前端代码中的 API_BASE
  4. 部署前端到 Netlify
  5. 测试完整的前后端通信
